Awesome Plugins 测试页面

说明：这是一个测试页面，演示如何加载和使用平安人寿的前端插件系统。

一、加载插件脚本

<!-- 测试环境 -->
<script src="https://stg1-public.cdn.lifeapp.pingan.com.cn/m/awesome-plugins/material-bootstrap/1.0.0/web/js/plugins/getPlugin/index.js"></script>

<!-- 生产环境 -->
<script src="https://public.cdn.lifeapp.pingan.com.cn/m/awesome-plugins/material-bootstrap/1.0.0/web/js/plugins/getPlugin/index.js"></script>

二、使用示例

;(async () => {
    // 获取插件
    const renderSatisfactionQuestionnaire = await window.AwesomePluginsBootstrap.plugins.getPlugin({
        pluginName: 'SatisfactionQuestionnaire',
        platform: 'web',
        env: 'prod'
    })

    // 挂载插件
    renderSatisfactionQuestionnaire({
        id: 'satisfaction-questionnaire',
        props: {
            sceneKey: 'xxx'
        }
    })()
})()
